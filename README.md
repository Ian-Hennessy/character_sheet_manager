# CATIMUS_PRIME's Character Sheet Manager 

This is a simple webpage for managing your character's sheet and levels, from level 0 to 20. There will be support for the following: 

All core classes in Dungeons and Dragons 5th Edition (2014)

Custom class creation with ability to input once, use forever for your Dungeons and Dragons characters 

Inventory management system with items from the DnD 5th Edition Player's Handbook
